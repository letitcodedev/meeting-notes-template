  ## Features

  - **Agenda Items** — Add timed agenda items with presenter and discussion notes
  - **Decisions Made** — Track decisions with context and rationale
  - **Action Items** — Assign tasks with owner, due date, priority, and status tracking
  - **Dark Mode** — Toggle between light and dark themes
  - **Print** — Clean print layout (hides toolbar)
  - **Export** — Download meeting notes as a JSON file

  ## Stack

  - [React](https://react.dev/) + [Vite](https://vitejs.dev/)
  - [Tailwind CSS](https://tailwindcss.com/)
  - [shadcn/ui](https://ui.shadcn.com/) components
  - [Lucide React](https://lucide.dev/) icons
  - [TypeScript](https://www.typescriptlang.org/)
  - [pnpm](https://pnpm.io/) monorepo

  ## Getting Started

  ```bash
  # Install dependencies
  pnpm install

  # Start the dev server
  pnpm --filter @workspace/meeting-notes run dev
  ```
